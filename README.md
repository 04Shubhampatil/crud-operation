# 🧩 Express CRUD API

A simple RESTful CRUD API built with **Node.js**, **Express**, and **MongoDB** using **Mongoose**.

---

## 🚀 Features

- Create (Add new user)
- Read (List all users / Get user by ID)
- Update (Modify user details)
- Delete (Remove user by ID)
- Error handling & status responses

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv
- body-parser
- cors
